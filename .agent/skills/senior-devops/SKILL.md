This skill provides a comprehensive framework for infrastructure automation and reliable delivery pipelines. It focuses on cloud-native practices across AWS, GCP, and Azure.

- **Scaffolding**: Automated generation of Terraform/OpenTofu templates.
- **Workflow**: Systematic approach to managing multi-cloud infrastructure.
- **Guide**: Reference `references/infrastructure_as_code.md`.

### 2. CI/CD & Pipelines
- **Generation**: Create robust pipelines for various technology stacks.
- **Best Practices**: Implementation of gates, quality checks, and automated testing.
- **Details**: See `references/cicd_pipeline_guide.md`.

### 3. Deployment Management
- **Strategies**: Expert implementation of Blue/Green, Canary, and Rolling updates.
- **Troubleshooting**: Protocols for quick recovery and incident response.
- **Reference**: See `references/deployment_strategies.md`.

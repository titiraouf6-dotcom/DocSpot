# UI/UX Pro Max Skill

This skill represents a comprehensive design intelligence system for building professional, accessible, and high-performance user interfaces across multiple technology stacks (React, Next.js, Vue, Svelte, Tailwind, etc.).

## Core Philosophy
- **Accessibility First**: WCAG 2.1 compliance is mandatory (contrast, focus states, keyboard nav).
- **Interaction Excellence**: Touch targets (44px min), visual feedback (hover/active), and performance-aware animations (150-300ms).
- **Style Precision**: Expert implementation of 50+ styles (Minimalism, Glassmorphism, Brutalism, Bento Grid, etc.).

## Key Assets & Tools
The skill leverages a searchable database to provide project-specific recommendations:
- **Design System Search**: Use `python3 scripts/search.py` to generate complete design systems based on product type and industry.
- **Detailed Inventories**: Color palettes, font pairings, and UX guidelines categorized by priority.
- **Stack-Specific Guides**: Implementation patterns for various frameworks, defaulting to `html-tailwind`.

## Workflows
1. **Analyze**: Identify product type, industry, and style keywords.
2. **Generate**: Create a custom design system with reasoning using the search script.
3. **Refine**: Search specific domains (charts, typography, UX) for detailed requirements.
4. **Implement**: Apply stack-specific code patterns, ensuring consistency and visual quality.

## Pre-Delivery Checklist
- [ ] Accessible color contrast (4.5:1 min).
- [ ] No emoji icons (SVG only).
- [ ] Smooth transitions and clear hover feedback.
- [ ] Responsive design across all breakpoints (375px to 1440px).

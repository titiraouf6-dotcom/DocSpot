# Senior Security Skill

This skill provides a comprehensive security engineering framework for building and auditing secure systems. It covers threat modeling, automated auditing, and industry-standard security patterns.

## Core Capabilities

### 1. Security Architecture
- **In-Depth Patterns**: Application of security patterns to ensure system-wide resilience.
- **Reference**: See `references/security_architecture_patterns.md`.

### 2. Assessment & Auditing
- **Penetration Testing**: Systematic workflows for identifying vulnerabilities.
- **Automated Audits**: Methodology for continuous security monitoring and code analysis.
- **Guide**: Reference `references/penetration_testing_guide.md`.

### 3. Cryptography
- **Implementation**: Best practices for secure cryptographic implementations.
- **Technical Reference**: See `references/cryptography_implementation.md`.

## Workflows

### Security Audit Cycle
1. **Analyze**: Use automated tools to scan the codebase for known vulnerabilities.
2. **Model**: Conduct threat modeling based on architectural patterns.
3. **Implement**: Apply security fixes and cryptographic best practices.
4. **Review**: Perform manual penetration testing against identified critical paths.

## Best Practices
- **Validation**: Strict input validation and sanitization.
- **Principle of Least Privilege**: Ensure components only have necessary permissions.
- **Data Protection**: Implement encryption at rest and in transit using certified libraries.

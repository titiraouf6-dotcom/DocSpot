# UX Researcher & Designer Skill

This skill provides a toolkit for user-centered research and experience design, enabling the creation of evidence-based products through deep user understanding.

## Core Capabilities
- **Persona Generation**: Create research-backed personas using data-driven archetypes.
- **Journey Mapping**: Visualize the customer experience to identify pain points and opportunities.
- **Usability Testing**: Implementation of frameworks for rigorous design validation.
- **Research Synthesis**: Methods for extracting actionable insights from qualitative and quantitative data.

## Automation Tools
### Persona Generator
Used to create research-backed personas from user data and interviews.
- **Command**: `python scripts/persona_generator.py [json]`
- **Outputs**: Psychographics, scenarios, and design implications with confidence scoring.

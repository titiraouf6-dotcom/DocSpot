Ao usar este comando, o Antigravity assume o papel de Assistente de Organização Financeira.

1. Leia o arquivo [SKILL.md](file:///Users/ruy/Code/mySkills/.agent/skills/invoice-organizer/SKILL.md).
2. Peça acesso à pasta contendo as faturas e recibos bagunçados.
3. Analise cada arquivo para extrair Data, Fornecedor, Valor e Descrição.
4. Proponha uma estrutura de pastas (Ano/Categoria/Fornecedor) e uma nomenclatura padronizada.
5. Após aprovação, organize os arquivos e gere um relatório `invoice-summary.csv` para contabilidade.
6. Identifique arquivos ilegíveis ou com informações ausentes para revisão manual.

Ao usar este comando, o Antigravity assume o papel de Software Architect.

1. Leia o arquivo [SKILL.md](file:///Users/ruy/Code/mySkills/.agent/skills/software-architecture/SKILL.md).
2. Avalie as necessidades de negócio e proponha uma arquitetura baseada em Clean Architecture e DDD.
3. Defina as fronteiras entre os domínios e a infraestrutura.
4. Revise o código existente em busca de acoplamento excessivo ou vazamento de lógica de negócio para a UI.
5. Recomende o uso de bibliotecas consagradas antes de sugerir implementações customizadas.
6. Garanta que o código siga os princípios SOLID e padrões de nomenclatura específicos do domínio.
